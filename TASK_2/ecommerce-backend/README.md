# Month 2 – E-Commerce Backend

## Project
Spring Boot E-Commerce backend with REST APIs for:
- Products
- Categories
- Users
- Orders

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Run Instructions
cd ecommerce-backend
mvn spring-boot:run

Base URL:
http://localhost:8080

H2 Console:
http://localhost:8080/h2-console
